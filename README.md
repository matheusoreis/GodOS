# Godot Online System - GodOS

O GodOS é uma base para quem quer criar seu próprio MMORPG usando a Godot. A ideia é dar aos desenvolvedores uma fundação simples e prática para construir seus próprios jogos, aproveitando as plataformas suportadas pela Godot.
